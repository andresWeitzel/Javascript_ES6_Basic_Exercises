### 1.0) Write a JavaScript Program to Copy a String to an Empty String [üîù](#index-)

<details>
  <summary>See solution</summary>
 <br>

#### Solution
 ```js
let emptyString = "";
let stringWithContent = "7623762736762367";
emptyString = emptyString + stringWithContent;

console.log(emptyString);
 ```

#### Console
 ```js
7623762736762367
 ```
